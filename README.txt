SATURATING BIT COUNTER SIMULATOR

This simiulator is designed for preliminary testing 
of a compiler assisted branch predictor that aids in 
choosing the number of bit used in a saturating bit
counter.

-----------------------------------------------------

To start the simulator run: SBC_Sim.exe

To make any modifications, refer to: SBMAin.c
